===========
LPC Vocoder
===========

This project implements a Linear Predictive Coding (LPC) vocoder, which includes
**encoder** and a **decoder** modules. The LPC vocoder uses standard LPC
calculation based on the `Wiener Filter <https://en.wikipedia.org/wiki/Wiener_filter>`_
and the `source-filter model <https://en.wikipedia.org/wiki/Source%E2%80%93filter_model>`_
